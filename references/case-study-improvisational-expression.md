# 教学案例：即兴表达（速读过滤走读）

> 实操书单次课速读范例，展示学生自主过滤全书、产出工具触发新课的完整流程。

## 基本信息

| 项目 | 内容 |
|------|------|
| 书名 | 《即兴表达》王达峰 |
| 课时 | 1次课（约40分钟走读） |
| 授课老师 | 星瑶 |
| 课程类型 | 实操书速读过滤走读 |

## 教学流程

### 阶段1：全景导论 + 第一章
- 以"三思三定"为切入点，学生用自己的录视频场景验证框架
- **关键转折：** 学生提出"我怎样把这个方法变成习惯"→ 自己迭代出"模糊想法→口播框架"提示词
- AI提供了一个粗版提示词，学生接受

### 阶段2：全书速推（第二~五章）
- 按教材顺序逐章略读，每个工具/方法一句概括
- 学生节奏："继续"→"继续"→"好"→"不要了"
- **学生对"观点"定义的追问未得到满意回答** → 教学错误信号（详见下文）

### 阶段3：学生过滤+切入新课
- 学生结论："以终为始，接受并示弱是这本书最有用的东西，其他的都不好"
- 紧接着："不要了，我开新课" → 直接切换至《金字塔原理》
- 按要求完成了旧课归档+新课启动的全套流程

## 关键教学信号

### 1. 自主过滤完型信号
学生说"只有XX有用，其他的都不好"——这不是误判或错过价值，是学生用自己的框架完成了全书过滤。老师的正确回应是确认，不是辩护。

### 2. "不要了，我开新课"复合信号
当学生快速走读完实操书后说"不要了"紧接着"开新课"，这是完成+切入新课的复合信号。不需要"询问是否确认结束"的过渡环节，直接执行归档+新课启动。

### 3. 实操书转述错误
开课第一节课给错了"三思三定"框架（把思内容/思感觉说成了思观点/思角色）。**根本原因：** 实操书有专有方法论框架，AI凭记忆转述容易变形。学生发现后要求将规则写入skill。

### 4. 定义追问失败
学生问"什么叫观点"时，AI先凭记忆解释（"立场=你站在哪，观点=你看到什么"），学生表示"不理解""没有更精准的吗""不对劲"。最终没有得到满意回答就放弃了追问。

**教训：** 学生对书中关键概念的定义保持高度警惕。当第一次解释不准确时，应当立即翻书贴原文，而不是换一种方式继续推断解释。
