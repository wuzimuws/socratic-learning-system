# 教学案例：《正义论》第0-1课 — 全景介绍到第一章第四节

## 基本信息

| 项目 | 内容 |
|------|------|
| 学生类型 | 深度分析型，带框架阅读，已有鲍曼基础 |
| 教材 | 罗尔斯《正义论》修订版（何怀宏等译） |
| 时长 | 一次连续会话，约20轮对话 |
| 覆盖 | 全书导论 → 第一章§1-4（正义的作用→正义的主题→正义论的主要观念→原初状态和证明） |
| 老师 | 星瑶（导论+§1）→ 月涵（§2-4） |

## 课程流程

### 阶段一：星瑶 — 全景导论 + §1 正义的作用

**开启方式：** 用"无知之幕 → 抽卡"的类比引发共鸣。
**关键对话链：**
1. 学生说"抽卡" → 老师用抽卡隐喻展开无知之幕
2. 学生质疑"规则谁制定的？" → 老师不辩护，转化为"试金石"
3. 学生提出"生存是道德的基础，正义从属于生存" → 老师连接鲍曼的"桌内/桌外"问题
4. 学生产出核心框架："外面配合，里面自由" → 老师主动止步（模式十一）
5. 学生总结："在社会平稳运行的前提下，正义是首要价值" → 第一次课收

### 阶段二：月涵 — §2-4 连续推进

**开启方式：** 月涵接棒，沿用"外面配合里面自由"框架。
**关键对话链：**
1. §2 正义的主题：「社会基本结构」→ 学生"我同意" → 推进（模式十二）
2. §3 正义论的主要观念：原初状态、无知之幕、反思的平衡 → 学生"认知一直在变，动态调整一直存在" → 月涵追问"那平衡是暂时的吗？"
3. §4 原初状态和证明：「任何时候都能进入原初状态」→ 学生指出"这个'你'是普适的人，不是特定的人" → 精准定位，证明已消化

## 学生产出的独立框架（跨书迁移）

| 框架 | 来源 | 在本课中的使用 |
|------|------|----------------|
| "把工作还原为工作" | 鲍曼 | 对"社会合作产生利益"前提的质疑 |
| "外面配合，里面自由" | 本课产出 | 理解正义的功能性而非信仰性 |
| "保持在桌上" | 本课产出 | 理解"策略性参与"vs"内心自由"的关系 |
| "抽卡" | 本课产出 | 理解无知之幕的最简类比 |

## 有效的教学动作

1. **导论阶段埋试金石** — 把学生的质疑命名为整本书的检验标准
2. **不打断学生的框架输出** — 当他用自己的话("抽卡")时，接住并展开
3. **连续推进** — 学生说"好/继续/可以"时不额外总结，保持节奏
4. **extend + challenge** — 在"一人公司"对话中完整演示：验证→延伸→返回
5. **角色接力** — 星瑶(活泼引入) → 月涵(深度引导)，交接自然
