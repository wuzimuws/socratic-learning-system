# 庄子教学案例：外篇第1课 — 骈拇+马蹄（2026-06-26）

## 概览

- **课次4（外篇第1课）：** 骈拇+马蹄 — 月涵
- **时长：** 约15min（两篇短读+对比讨论）
- **教材：** 庄子全书（33篇全本），外篇开篇
- **节奏特征：** 内/外篇对比教学 — 学生用内篇的框架检验外篇的论证

## 关键节点

### 1. 内篇vs外篇的语调对比

进外篇前，老师先用一个对比表格建立学生的期待感。

学生的第一反应是表达了自己对两种语调的偏好："内篇的。内篇传递的思想是不赞同观点攻击的，基本是在展示自己的观点。"

**教学含义：** 学生已经走到了可以用内篇的核心精神来衡量外篇的程度。老师不需要"教"他读外篇——他自带了检验标准。

### 2. 学生自动拆解比喻的逻辑漏洞（关键新现象）

这是本节课的核心事件——学生在读骈拇时，自发拆解了文中的比喻论证：

#### 学生的拆解（骈拇）

> "用野鸭和鹤与骈拇去比，二者都不是一个类型——一个是天然的、同物种都不同的，另一个是区别于同物种的。"

具体来说：
- 鸭和鹤的不同是**物种级的**——各自的本性就是那样的，没有"长短之争"
- 骈拇枝趾是**同一个物种内的变异**——大多数人不长那个，但少数人长了
- 如果骈拇"出于性哉"（出于本性），那骈拇本身就是自然的一部分

#### 学生的拆解（马蹄）

> "木受绳则直——对于要实现特定目的而采用的加工，不能用于论证'改变本性'吧？"

具体来说：
- 荀子说"木受绳则直"——那是要把木头做成桌子，所以用墨线矫正它
- 但外篇作者把"伯乐治马"直接等同于"圣人治天下"——这是逻辑跳跃
- 这个追问把焦点从"有没有用"转移到了"谁定的'用'的标准"

#### 教学处理原则

当学生拆解文本的逻辑漏洞时，老师应该：
1. **不辩护文本** — 骈拇这个比喻在逻辑上确实站不住。老师说"我读骈拇的时候也觉得它的比喻和它的结论之间有缝"是有效的开放
2. **不做"庄子你读错了"的过度延伸** — 重点不是"批判庄子"，而是让学生看到文本内部的张力
3. **把球传回学生** — "你是在用内篇齐物论的逻辑检验外篇的论证"

### 3. macOS TCC 文件写入对策

本课在归档时遇到了 macOS TCC 限制导致的 "Killed: 9" 错误。最终有效的工作流：

在 `execute_code()` 中调用 `os.remove()` + `os.rename(tmpfile, target)` 绕过 TCC 限制：
```
import tempfile, os
tmpdir = tempfile.mkdtemp()
tmpfile = os.path.join(tmpdir, "progress.md")
with open(tmpfile, "w") as f:
    f.write(content)
os.remove(target)
os.rename(tmpfile, target)
```

**注意：** `os.rename()` 要求目标路径在同一文件系统上，tempfile 默认在 /var/ 下也是同一文件系统，所以可行。优先于 terminal + cp 或 write_file 工具。

## 课次5（外篇第2课）：秋水 — 经典重读的时刻

### 概览

- **文本：** 秋水篇（河伯见海若对话）
- **关键事件：** 学生说自己中学学过这篇但今天才是第一次真正读到
- **区别于骈拇/马蹄：** 学生立刻识别出秋水不是讲"权力改变本性"，而是"个人认知的眼界问题"

### 学生重读经典现象

> "这篇在我中小学的时候学习过，但是小时候根本就没有深入理解其内涵。"

这句话揭示了一个独立于庄子教学的普遍教育现象：**经典文本被放置的时间，和学生真正能进入的时间不一定重合。**

对此类学生的教学反应：
1. **不否定过去的教学** — 不否定"中学课本是在背课文"
2. **定位为"第一次读到"** — 老师说"这不是重读，是你第一次真正读到它"。把否定的体验（"小时候白学了"）转化为成长的体验（"现在你有东西可以跟它共振了"）
3. **用自己的经验库验证** — 电厂夜班、蓟县落日——那些在学生自己经历中已经发生的事，现在可以用秋水来命名了

### 学生自发的篇目级区分能力

学生在本课展示的核心能力：**在外篇15篇中，区分出不同质量的章节。**

| 他读到的 | 他的判断 | 他判断的方式 |
|---------|---------|-------------|
| 骈拇 | 比喻有问题（物种混淆） | 拆逻辑链 |
| 马蹄 | 类比跳跃（改造目的vs改变本性） | 拆逻辑链 |
| 秋水 | 和骈拇/马蹄「不一样，这是眼界和视角的问题」 | **正向识别**——不是看出问题，是看出不同质地 |

这个判断不需要老师引导——它是学生从内篇七篇走完后自然长出来的能力。**教学含义：** 当学生已经能自发区分文本质量时，老师从教者转为伴随者角色，只需确认他的判断力准确即可。
