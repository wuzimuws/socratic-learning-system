# 庄子教学案例：第0-1次课（2026-06-25）

## 概览

- **教材：** 庄子全书（33篇全本 + 南怀瑾附录），719页PDF
- **原教材：** 选编本（三秦出版社，10篇精选）
- **学生纠正：** "精选的版本不好，用这个全书替代"
- **节奏：** 一次会话中完成了全景导论 + 逍遥游解读 + 齐物论开篇 + 养生主过渡
- **老师：** 星瑶（开篇）

## 关键节点

### 1. 教材选择——学生偏好全书而非选本

学生明确拒绝了精选版本（10篇选编本），要求替换为33篇全本。这是重要的信号：**此学生不满足于入门导读，他想要面对整部经典。**

**后续操作：** 替换 material/ 中的PDF文件，重新生成完整的 curriculum.md（23课）和 config.md（含全部33篇的介绍和南怀瑾附录）。

### 2. 全景介绍的连接方式

不同于鲍曼/罗尔斯/GTD的社会科学框架，庄子的导论使用了**四条贯穿全书的思想线索**来组织：

1. **相对主义** — 瓦解"唯一正确"（逍遥游、齐物论、秋水）
2. **无用之用** — 反功利主义（逍遥游结尾、人间世、山木）
3. **生死一如** — 不恐惧死亡（养生主、大宗师、至乐）
4. **批判** — 对仁义礼乐和社会制度的激烈拆解（胠箧、马蹄、盗跖）

以及**外篇 vs 内篇的张力**（内篇圆融、外篇激烈）作为课外遗留问题。

**效果：** 学生立即抓住"视角转换"作为自己的入口——这个连接方式是有效的。

### 3. 学生的自发连接链

本课最显著的特征是学生的连接能力。以下是他的完整思维轨迹：

```
视角转换（自己的体验）
  → 至人无己（原文概念）
    → 佛教无我（跨传统连接）
      → 薛定谔的猫（跨学科连接——量子物理）
        → 是名XX，非XX（跨传统连接——金刚经）
          → 如筏喻者（跨传统连接——金刚经另一概念）
            → 追问到最后成了混沌（走到语言的边界）
              → 这个状态很难受（表达不适）
                → 庖丁解牛作为落地（接受具体胜过抽象）
```

这是一个**从个人体验到个人哲学的完整路径**，中间经过了：文本概念 → 跨传统比较 → 跨学科类比 → 语言边界意识 → 不适表达 → 具体落地。老师在这个路径中的角色是**伴随者而非领导者**——每一次学生做出连接，老师做的是确认 + 指出差异 + 回到文本，而不是打断或纠正。

### 4. 庄子教学的独特性对照

| 维度 | 鲍曼教学 | 罗尔斯教学 | GTD教学 | 庄子教学 |
|------|---------|-----------|---------|---------|
| 学生入口 | 工作伦理批判（共鸣） | 正义感（理性推演） | 效率焦虑（工具诊断） | 视角转换（个人体验） |
| 学生主要动作 | 联系个人经验 | 逻辑推演、质疑前提 | 动手改造工具 | 跨传统连接、自我追问 |
| 老师主要动作 | 苏格拉底追问 | 浓缩结构教学+框架转换 | 教练式诊断+现场应用 | 拆解逻辑骨架+跨域类比 |
| 学生产出 | 生活道德框架 | "抽卡"类比 | 系统改造+原则浓缩 | "人与宇宙本就是一体" |
| 危险信号 | 学生说"我懂了"但不深入 | 学生说"还是不理解" | 学生说"有道理"但不动手 | 学生说"太飘了/抓不住" |

### 5. 教材处理注意事项

本教材（庄子全书PDF）有完整的书签/目录，但目录层级较深（L0=卷，L1=内/外/杂篇，L2=各篇）。需要特殊处理：
- 使用 pypdfium2 的 get_toc() 方法提取（返回的是 generator，需要先 list()）
- 内7外15杂11，共33篇
- 附录为南怀瑾讲《庄子》8讲，按主题编排而非按篇目编排
- 教材本身有"分节导读""品庄悟道"等注释板块，适合教学使用
