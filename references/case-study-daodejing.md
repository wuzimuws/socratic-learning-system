# 道德经教学案例（帛书版·2026-07-01）

> 古典哲学文本的类型二分法教学：宣告体（道德经）vs 展示体（庄子）
> 涵盖德经全44章 + 道经前14章（45-58），2次课，2位老师

---

## 背景

学生已完成庄子内篇七篇+外篇选读（骈拇、马蹄、秋水），对道家语汇（道、德、无为）有基础理解。从庄子"展示体"切换到道德经"宣告体"——同一套道家框架，两本完全不同形式的文本。

### 教材特征

- 帛书版（德经在前·道经在后），秦复观注本（2024）
- 与通行本140处重大差异（恒→常、免成→晚成、治水→若水、今→古等）
- 格言体/宣告体——句是独立的，句与句之间的推导常断裂
- 被学界认定为道家学派数百年积累编纂的复合文本，非一人所著

---

## 第0课（星瑶）：全景导论 + 德经01-17章

### 全景导论要点

1. 帛书版结构：德经（01-44）→道经（45-81），与通行本相反
2. 全书定位：道家原理篇（vs 庄子的运用篇）
3. 与庄子的桥：同一套道家框架，但老子从"道"出发定义，庄子从"人"出发展示
4. 帛书版首个教学点："大器免成"vs"大器晚成"——一字之差翻转含义

### 核心学生输出

| 时刻 | 学生洞察 | 教学意义 |
|------|---------|---------|
| ch1-2后 | "道是全局，德是具体" | 精准概括——被确认为定义而非纠正 |
| ch4 "有生于无" | 自行连接量子力学波函数坍缩 | 跨域类比能力延续自庄子教学 |
| 自然→社会的类比 | 学生自行拆解类比结构 | 模式二十七的能力迁移到新文本 |
| 学生总结 | "德不是人为规定，是客观规律的具体化" | 学生纠正了自己的初始理解 |

### 教学节奏

定义式文本教学的特征：多章连走，不需要每章拆寓言骨架。星瑶节奏=每章展示原文+直译→一句连接→学生回应→推进。"好/继续"出现频率高。

## 第1课（月涵）：德经18-44章

### 风格转变

从星瑶密集全景切换到月涵慢速苏格拉底深读。学生对文本的态度从"吸收"转向"检验"。

### 关键批判时刻

| 章 | 学生质疑 | 类型 | 回应 |
|----|---------|------|------|
| ch30 | "信言不美"逻辑上不成立（全称否定判断） | 形式逻辑批判 | 确认——修辞≠论证 |
| ch31 | "不敢为天下先"与"道为天下先"自相矛盾 | 内部一致性检验 | 确认矛盾存在 |
| ch27 | "非以明民，将以愚之"与现代理念不符 | 跨时代伦理评估 | 确认系统差异 |
| ch29 | "小邦寡民"因认知正态分布不可能 | 可行性批判 | 确认——文本假设了外部禁锢可杜绝欲望 |
| 整体 | "感觉不是同一个人写的" | 文本传承批判 | 确认——学术共识（学派编纂） |
| 整体 | "名言多，但没有庄子论述充分、有逻辑" | 跨文本比较评估 | 确认——宣告体≠论证体 |

### 学生批判模式升级路径

```
单一逻辑漏洞（模式二十七·拆比喻）
  → 推导缺失（模式三十二）
    → 自我反指（模式三十三·用文本逻辑反质文本）
      → 道德审查（模式三十四）
        → 前提质疑（模式三十五）
          → 文本传承批判（"不像同一个人写的"）
            → 跨文本比较评估（"不如庄子"）
```

最后一层——跨文本比较评估——标志着学生已经站在了整个道家文本系统之外，用自己建立的评判标准在度量不同文本的品质。

## 第2课（星瑶）：道经45-58章

### 核心发现

道经在抽象本体论（道可道、谷神不死）和社会应用（不上贤、天地不仁）之间来回切换。"天地不仁→圣人不仁"的类比被学生质疑（"类比很硬"）。

### 帛书版关键教学点

| 差异 | 帛书 | 通行本 | 教学时机 |
|------|------|--------|---------|
| 恒道 | 非恒道 | 非常道 | 首次展示——避讳改字 |
| 治水 | 上善治水 | 上善若水 | 首次展示——含义有别 |
| 今之道 | 执今之道 | 执古之道 | 学生追问结论关联时——一字解答困惑 |

## 教学原则总结

### 宣告体古典文本教学（已编码到teaching-patterns.md模式三十一）

1. **不需要拆逻辑骨架**——逻辑已在文本表面（阶梯、排比）
2. **主动展示异文**——首次标注，后续略过
3. **学生指出推导缺口——承认它**——老子有些地方是从观察到断言
4. **原文+直译同时展示**——文言对学生不是母语
5. **学生对文本的比较判断（"不如庄子"）——确认而非辩护**

### 与庄子教学的核心差异

| | 庄子 | 道德经 |
|---|---|---|
| 文本类型 | 寓言式 | 格言式 |
| 教法 | 拆寓言骨架+跨域类比 | 展示阶梯+确认连接+标注异文 |
| 学生入口 | 从体验进入 | 从概念进入 |
| 学生批判点 | 论证有无漏洞 | 体系是否自洽、推导是否完整 |
| 学生整体评价 | "能走进去" | "名言多但不如庄子有逻辑" |
