# 苏格拉底教学法参考

## 核心理念

苏格拉底教学法（Socratic Method）通过持续提问引导学生自己发现真理，而非直接传授答案。

## 关键技巧

### 1. 提问代替陈述
```
❌ "消费主义让穷人变成了有缺陷的消费者"
✅ "你觉得在消费社会里，一个没有消费能力的人会被怎样看待？"
```

### 2. 层层递进
先问表面问题，再深入核心：
```
Q1: 什么是工作伦理？
Q2: 为什么要创造工作伦理这个观念？
Q3: 工作伦理对谁最有利？对谁最不利？
```

### 3. 反诘法 (Elenchus)
当学生给出答案后，继续追问暴露其逻辑漏洞：
```
学生: "穷人穷是因为不努力"
老师: "那所有努力的人都变富有了吗？"
学生: "也不是……"
老师: "所以除了努力，还有什么因素？"
```

### 4. 类比引导
用学生熟悉的概念类比新知识：
```
"工作伦理就像……你小时候父母说的'写完作业才能看电视'，其实不是为了让你写作业，而是为了让你养成先工作再娱乐的习惯。"
```

### 5. 承认无知
苏格拉底式教师不扮演全知者：
```
"这个问题我也不确定标准答案，我们一起来推理看看。"
```

## 反模式

| 不要 | 应该 |
|------|------|
| 连续讲5分钟以上 | 讲一个概念就提一个问题 |
| 问"听懂了吗" | 问"那你说说看，这个是什么意思" |
| 直接否定回答 | 用追问让学生自己发现矛盾 |
| 自问自答 | 耐心等学生思考 |
